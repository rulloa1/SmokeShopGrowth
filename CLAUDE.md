# Claude Code Project Guidelines - Google Maps Lead Scraper

## Project Overview
A tool for scraping business leads from Google Maps, with automation and monetization in focus.

## Tech Stack
- Backend: Python (FastAPI/REST)
- Frontend: Vanilla HTML/JS/CSS (as per global rules)
- Scraper: Node.js (Playwright/Puppeteer potential)
- Deployment: Docker, Railway, Render

## Development Commands
- `npm start`: Run the web server (server.js)
- `python main.py`: Run the main scraping logic
- `npm install`: Install dependencies
- `pip install -r requirements.txt`: Install Python dependencies

## Code Style & Rules
- Follow "Builder Mindset" (Speed, Automation, Monetization).
- Use Vanilla CSS for UI (Premium, Modern aesthetics).
- Automate repetitive tasks.
- Prioritize small business niches (Smoke shops, Vape shops, etc.).
